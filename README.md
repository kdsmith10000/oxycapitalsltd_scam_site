# OxyCapitals Scam Exposure Site

This website exposes the cryptocurrency investment scam operated through **oxycapitalsltd.com**.

## Evidence Documented

- **Blockchain transaction analysis** tracing stolen funds
- **Scam wallet addresses** (Bitcoin, Ethereum, USDT)
- **Money flow hierarchy** showing how funds move from victims to the scammer to exchanges
- **Screenshots** of the fake platform and impersonator accounts

## Deployment

This site is built with Next.js and designed to be deployed on Vercel.

### Deploy to Vercel

1. Push this repository to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Import your GitHub repository
4. Deploy (Vercel will auto-detect Next.js settings)

### Local Development

```bash
npm install
npm run dev
```

## Purpose

This site exists to:
1. Warn potential victims before they send money
2. Document evidence for law enforcement
3. Help victims report the scam to authorities

## Reporting

If you've been scammed:
- **FBI IC3:** https://ic3.gov
- **FTC:** https://reportfraud.ftc.gov
- Report wallet addresses to cryptocurrency exchanges
