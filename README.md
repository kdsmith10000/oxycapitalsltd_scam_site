# Crypto Scam Exposure Network

This website exposes cryptocurrency investment scams including **OxyCapitalsLtd.com** and **MirrorExp.com**.

## Scams Documented

### 1. OxyCapitals (oxycapitalsltd.com)
- Discord admin impersonation scam
- **Total Stolen:** $40,400+
- Status: Active

### 2. MirrorExp (mirrorexp.com) - NEW
- **Typosquatting attack** - scammer creates usernames nearly identical to real admins
- Discord admin impersonation using typosquatted usernames
- Multi-chain operation (BTC, ETH, USDT, USDC)
- **Total Stolen:** $22,000+
- **Operation Duration:** 6+ months
- Status: Active

## Evidence Documented

- **Blockchain transaction analysis** tracing stolen funds across multiple chains
- **Scam wallet addresses** (Bitcoin, Ethereum, USDT, USDC)
- **Money flow hierarchy** showing how funds move from victims to the scammer to exchanges
- **Screenshots** of the fake platforms and impersonator accounts
- **Typosquatting evidence** showing how scammers mimic real usernames

## Deployment

This site is built with Next.js and designed to be deployed on Vercel.

### Deploy to Vercel

1. Push this repository to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Import your GitHub repository
4. Deploy (Vercel will auto-detect Next.js settings)

### Local Development

```bash
npm install
npm run dev
```

## Purpose

This site exists to:
1. Warn potential victims before they send money
2. Document evidence for law enforcement
3. Help victims report the scam to authorities

## Reporting

If you've been scammed:
- **FBI IC3:** https://ic3.gov
- **FTC:** https://reportfraud.ftc.gov
- Report wallet addresses to cryptocurrency exchanges
